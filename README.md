# Gang Manager

Aplikacja do zarządzania wymianami kart w gangu w grze The Gang.

## Wgranie na Vercel (darmowo)

1. Wgraj wszystkie pliki na GitHub do repozytorium `gang-manager`
2. Wejdź na vercel.com → "Add New" → "Project" → wybierz repo
3. Kliknij "Deploy"
4. Gotowe — dostaniesz link typu `gang-manager.vercel.app`

## Logowanie

- **Admin:** login `admin`, hasło `Twojastara00`
- **Zastępca:** login `zastepca`, hasło `Twojastara00`
- **Członek:** dowolny nick (min 2 znaki), bez hasła

## Baza danych

Aplikacja zapisuje dane w Firebase Firestore (projekt `gang-wymiana`).
Wszyscy widzą te same dane na żywo.
